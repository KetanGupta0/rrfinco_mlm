A complete, production-ready Multi-Level Marketing (MLM) platform built with PHP, MySQL, and modern web technologies. Features tiered daily cashback, 30-level commission structure, bonus systems, and comprehensive admin panel.

## 🌟 Key Features

### 💰 Financial Engine
- **Tiered Daily Cashback**: 0.10% to 0.16% daily based on investment amount
- **30-Level Commission Structure**: 25% down to 1% across 30 levels
- **Bonus 1**: 20% extra for maintaining ₹2,000+ balance for 30 days
- **Bonus 2**: 10-20% extra for weekly business challenges (₹50k/₹1L)
- **Investment Plans**: Flexible investment management system

### 👥 User Management
- **User Registration**: With sponsor referral system
- **Authentication**: Secure login with session management
- **Profile Management**: Complete user profile and banking details
- **Wallet System**: Real-time balance tracking and transactions
- **Withdrawal Management**: Secure payout request system

### 📊 Analytics & Reporting
- **Dashboard**: Comprehensive financial overview with charts
- **Transaction History**: Complete audit trail of all transactions
- **Team Management**: Downline tree visualization and statistics
- **Earning Reports**: Detailed breakdown of income sources
- **Badge System**: Achievement tracking and gamification

### 🔧 Admin Panel
- **User Management**: Complete user administration
- **Financial Oversight**: Monitor all transactions and payouts
- **System Configuration**: Platform settings and maintenance
- **Automated Processing**: Cron job management and monitoring

### 📧 Communication
- **Email System**: SMTP support with professional templates
- **Password Reset**: Secure token-based password recovery
- **Notifications**: Automated email notifications for key events

## 🚀 Quick Start

### Prerequisites
- **PHP 8.0+** with PDO extension
- **MySQL 5.7+** or **MariaDB 10.0+**
- **Apache/Nginx** web server
- **Composer** (optional, for dependency management)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/bachat-pay-mlm.git
   cd bachat-pay-mlm
   ```

2. **Setup Web Server**
   ```bash
   # Place files in web root (e.g., XAMPP htdocs)
   # Example: C:\xampp\htdocs\bachat-pay-mlm\
   ```

3. **Database Setup**
   ```sql
   -- Create database
   CREATE DATABASE bachatpay_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

   -- Import schema
   mysql -u root bachatpay_db < database.sql
   ```

4. **Configuration**
   Edit `includes/config.php`:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_USER', 'root');          // Your MySQL username
   define('DB_PASS', 'your_password'); // Your MySQL password
   define('DB_NAME', 'bachatpay_db');

   // Email Configuration (for production)
   define('MAIL_ENABLED', true);
   define('SMTP_HOST', 'smtp.gmail.com');
   define('SMTP_PORT', 587);
   define('SMTP_USERNAME', 'your-email@gmail.com');
   define('SMTP_PASSWORD', 'your-app-password');
   ```

5. **Create Required Directories**
   ```bash
   mkdir -p logs
   chmod 755 logs
   ```

6. **Access Platform**
   ```
   URL: http://localhost/bachat-pay-mlm/
   Default Admin: Check database.sql for admin credentials
   ```

| Investment Range | Daily Rate | Monthly Rate | Example (₹100k) |

### Bonus Systems

#### Bonus 1 (30-Day Maintenance)
- **Requirement**: Maintain ₹2,000+ wallet balance for 30 consecutive days
- **Reward**: 20% extra on monthly cashback earned
- **Example**: ₹4,200 monthly cashback + ₹840 bonus = ₹5,040

#### Bonus 2 (Weekly Challenge)
- **₹50,000 Business**: 10% bonus on daily cashback
- **₹100,000+ Business**: 20% bonus on daily cashback
- **Period**: Weekly (Monday to Sunday)
- **Source**: Business generated by downline members

## 🔄 Automated Processes (Cron Jobs)

### Daily Cashback Processing
```bash
# Process daily earnings for all active investments
curl "http://yoursite.com/cron.php?action=daily_cashback"
```
- Runs daily at midnight
- Credits wallet balances
- Records transactions
- Distributes upline commissions

### Badge Assignment
```bash
# Award achievements to qualifying users
curl "http://yoursite.com/cron.php?action=assign_badges"
```
- Runs daily
- Checks user progress against badge criteria
- Awards new badges automatically

### Bonus Processing
```bash
# Process monthly Bonus 1
curl "http://yoursite.com/cron.php?action=bonus_1"

# Process weekly Bonus 2
curl "http://yoursite.com/cron.php?action=bonus_2"
```
